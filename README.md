# 📒 Contact Book

A simple Contact Book application built using Python and the BeautifulTable library. It allows users to store, search, edit, delete, and view contacts through a command-line interface (CLI).

## 🚀 Features

- Add a new contact
- Delete an existing contact
- Edit contact information
- Search contacts
- View all contacts
- Display contacts in a formatted table
- Prevent duplicate phone numbers

## 🛠 Technologies Used

- Python 3
- BeautifulTable

## 🔍 Search

Search works using any keyword.

Example:

Search: john

Search: pune

Search: gmail

You can also sort the result by:

- name
- address
- phone_number
- email

## Example

Name: John

Address: Mumbai

Phone: 9876543210

Email: john@gmail.com

## Output

+----+------+---------+---------------+
| No | Name | Address | Phone | Email |
+----+------+---------+---------------+
| 1  | John | Mumbai  |9876543210|john@gmail.com|
+----+------+---------+---------------+
